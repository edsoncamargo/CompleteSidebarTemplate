Sidebar

The Angular Sidebar component is a user interface element designed to create side navigation menus in web applications. It provides an expandable or collapsible sidebar that contains links and navigation options for quick access to different sections or features of the application.

Usage

Module Import

Import the module of the component in your app.component.ts.

Component Usage

If you want to use the methods of the service, inject the SidebarService dependency.

Template Usage

Use the ec-ns-sidebar selector to use the component, and add your custom content inside the selector.

Selector Description
options This is where you will add all the custom HTML inside the Sidebar.

@Input() Properties

Property Type Default Value Description
startsOpen boolean true Starts the component in the open state.
