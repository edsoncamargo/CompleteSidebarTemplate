Navbar

The Angular Navbar component is a versatile navigation element designed to provide an optimal user experience across various devices. This responsive navbar seamlessly adapts its layout, transitioning to a hamburger button on mobile devices for improved usability.

Usage

Module Import

Import the module of the component in your app.component.ts.

Template Usage

Use the ec-ns-navbar selector to use the component, and add your custom content inside the selector.

Selector Description
desktop Here you will add all navigation content for the desktop resolutions of the navbar.
mobileBrand Here you have the option to add something related to your brand, which will be fixed on the left side of the navbar for mobile resolutions.
mobile Here you will add all navigation content for mobile resolutions of the navbar, which will be inside the modal that opens when clicking the hamburger button.
