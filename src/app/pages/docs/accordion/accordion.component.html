Accordion

The Angular Accordion component is a user interface element designed to manage and display collapsible content sections. It allows users to organize information in a structured and space-efficient manner, presenting a clear and user-friendly interface.

Usage

Module Import

Import the module of the component in your app.component.ts.

Component Usage

If you want to use the methods of the service, inject the AccordionService dependency.

Template Usage

Use the ec-ns-accordion selector to use the component, and add your custom content inside the selector.

Selector Description
title This is where you will add a title custom HTML inside the Accordion header.
content This is where you will add all the custom HTML content inside the Accordion content.

@Input() Properties

Property Type Default Value Description
isOpen boolean false You can open and close component by isOpen property too.
